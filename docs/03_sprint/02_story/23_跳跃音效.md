# 跳跃音效

作为 一名玩家
我想要 跳跃时有相应的音效反馈
以便 获得更好的操作感和游戏体验

---

## 验收标准（AC）
1. 普通跳跃播放"呼"的起跳声（0.2秒）
2. 二段跳播放更清脆的"嗖"声（0.2秒）
3. 着陆时播放轻微的"哒"声（0.1秒）
4. 不同表面跳跃音效略有不同（草地vs石块）
5. 快速连续跳跃时音效不会重叠
6. 跳跃音效音量可以根据角色高度略有变化

---

## 备注
- **颗粒度**：1人日
- **美术需求**：SFX（起跳声、二段跳声、着陆声、不同材质声）
- **技术实现**：使用AudioStreamPlayer2D，根据表面类型选择不同音效
- **音效细节**：
  - 普通跳跃：低音呼啸声，强度中等
  - 二段跳：高音嗖声，更清脆
  - 草地着陆：柔软的"噗"声
  - 石块着陆：清脆的"嗒"声
- **音效参数**：
  - 起跳：0.15-0.25秒，音量-3dB
  - 着陆：0.1秒，音量-6dB
  - 二段跳：0.2秒，音量0dB，音高+2半音
- **防重叠机制**：设置最小播放间隔50ms
- **动态音量**：根据跳跃高度调整音量（高度越高音量越大）
- **性能要求**：音效加载延迟<20ms
- **文件格式**：OGG Vorbis，压缩率适中
- **音效管理**：预加载所有跳跃音效，避免运行时加载
