# 玩家生命值系统

作为 一名玩家
我想要 拥有生命值系统来承受伤害
以便 增加游戏的挑战性和策略性

---

## 验收标准（AC）
1. 玩家初始生命值为3点（显示为3颗心）
2. 被敌人触碰时扣除1点生命值
3. 受伤后有1.5秒的无敌时间（角色闪烁）
4. 生命值为0时游戏结束，显示"游戏结束"画面
5. 收集水果可以恢复1点生命值（最多恢复到3点）
6. 生命值变化时HUD实时更新显示

---

## 备注
- **颗粒度**：1.5人日
- **美术需求**：PX（心形图标、空心图标）、ANI（角色受伤闪烁动画）、UI（生命值显示界面）
- **技术实现**：添加health变量，使用信号系统通知HUD更新
- **无敌机制**：使用Timer节点，受伤期间disable_collision
- **视觉反馈**：受伤时角色闪烁，屏幕震动效果
- **游戏结束**：生命值为0时加载游戏结束场景
- **HUD集成**：与HUD系统通过信号通信
- **参数配置**：max_health = 3，invincible_time = 1.5
- **音效**：SFX（受伤声、恢复生命声、游戏结束声）
- **性能要求**：无敌检测准确，HUD更新延迟<100ms
