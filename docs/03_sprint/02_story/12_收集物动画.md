# 收集物动画

作为 一名玩家
我想要 看到收集物有生动的动画效果
以便 增加游戏的视觉吸引力和收集乐趣

---

## 验收标准（AC）
1. 金币持续旋转并有上下浮动效果（浮动幅度20像素）
2. 水果有轻微的左右摇摆和呼吸缩放效果
3. 收集物周围有柔和的光晕效果
4. 收集时播放收集动画（缩小消失+粒子爆发）
5. 不同收集物有不同的动画速度
6. 收集物在被收集前有预警闪烁（0.5秒）

---

## 备注
- **颗粒度**：1.5人日
- **美术需求**：ANI（旋转、浮动、摇摆、缩放、收集动画）、VFX（光晕效果、粒子特效）
- **技术实现**：使用AnimationPlayer和Tween节点
- **动画细节**：
  - 金币：360度旋转/2秒，正弦浮动/3秒
  - 水果：±5度摇摆/2秒，1.1-1.0缩放/2秒
- **光晕系统**：使用Light2D或shader实现柔和光晕
- **粒子预设**：金币（金色星形）、水果（彩色花瓣）
- **性能优化**：使用对象池管理粒子，限制同时播放的动画数量
- **参数配置**：float_amplitude = 20，rotate_speed = 180，collect_duration = 0.3
- **性能要求**：每屏最多30个动画，60FPS稳定运行
