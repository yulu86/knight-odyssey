# 物理重力系统

作为 一名玩家
我想要 角色受真实的物理重力影响
以便 游戏操作更加自然和符合直觉

---

## 验收标准（AC）
1. 角色在空中会持续受到向下的重力加速度
2. 重力加速度为1200像素/秒²
3. 角色下落时有相应的下落动画
4. 下落速度有最大限制（terminal velocity）
5. 重力系统能与跳跃和移动系统完美配合
6. 角色从平台边缘掉落时有短暂的滞空时间

---

## 备注
- **颗粒度**：1人日
- **美术需求**：ANI（下落动画、边缘滑落动画）
- **技术实现**：使用Godot内置的物理系统，调整gravity_scale参数
- **物理参数**：gravity = 1200，terminal_velocity = 800
- **碰撞检测**：使用move_and_slide()处理地面碰撞
- **边缘处理**：添加coyote_time参数（0.1秒）
- **性能要求**：60FPS，物理计算不影响游戏性能
- **调试工具**：显示实时速度和重力状态的调试信息
