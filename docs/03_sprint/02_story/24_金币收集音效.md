# 金币收集音效

作为 一名玩家
我想要 收集金币时有令人满足的音效
以便 增强收集的成就感和游戏乐趣

---

## 验收标准（AC）
1. 收集金币播放清脆的"叮"声（音高C6，0.15秒）
2. 连续快速收集时音调逐渐升高
3. 金币收集音效有轻微的回响效果
4. 收集多个金币时有和弦叠加效果
5. 音效不会因为频繁收集而产生疲劳感
6. 收集特效和音效完美同步

---

## 备注
- **颗粒度**：0.5人日
- **美术需求**：SFX（金币收集声、连收集声、和弦声）
- **技术实现**：使用AudioStreamPlayer2D，实现音高变化和回响
- **音效设计**：
  - 单个金币：正弦波+方波混合，C6音高
  - 音高变化：每次收集+1半音，最高到C7
  - 回响效果：使用音频总线添加reverb
  - 和声收集：同时收集多个金币触发
- **音效参数**：
  - 基础音高：C6（523Hz）
  - 持续时间：0.15秒
  - 音量：-2dB
  - 回响时间：0.3秒衰减
- **高级特性**：
  - 音量渐变：根据收集速度调整
  - 音色变化：偶尔使用不同音色
  - 3D效果：使用立体声定位
- **性能优化**：
  - 音效对象池管理
  - 限制同时播放数量（最多5个）
- **文件格式**：OGG，44.1kHz，16bit
- **测试要求**：快速连收10个金币不产生音频 clipping
