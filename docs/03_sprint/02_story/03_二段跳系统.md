# 二段跳系统

作为 一名玩家
我想要 在空中能够再次按下跳跃键进行二段跳
以便 能够到达更高的平台和躲避复杂的地形

---

## 验收标准（AC）
1. 当我在第一次跳跃后再次按下跳跃键时，执行二段跳
2. 二段跳高度为150像素，略低于第一次跳跃
3. 每次落地后重置二段跳次数（最多可二段跳一次）
4. 二段跳有独特的动画效果，区别于普通跳跃
5. 二段跳时有粒子特效反馈
6. 二段跳过程中仍可进行左右移动

---

## 备注
- **颗粒度**：1.5人日
- **美术需求**：PX（角色精灵图）、ANI（二段跳动画帧）、VFX（二段跳粒子特效）
- **技术实现**：添加jump_count变量，使用is_on_floor()检测落地
- **输入处理**：复用InputAction的jump动作，增加计数逻辑
- **物理参数**：二段跳velocity = -450
- **状态管理**：使用状态机管理地面、空中、二段跳状态
- **性能要求**：60FPS，二段跳响应延迟<50ms
