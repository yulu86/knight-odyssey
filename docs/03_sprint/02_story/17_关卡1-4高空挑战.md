# 关卡1-4高空挑战

作为 一名玩家
我想要 在一个高空的关卡中考验跳跃勇气和精准度
以便 体验刺激的高空冒险并提升跳跃技巧

---

## 验收标准（AC）
1. 关卡垂直高度为400米，以垂直攀登为主
2. 包含20个悬浮平台，部分有风的影响
3. 设置3个上升气流区域，可将角色托起
4. 有8个绿色史莱姆放置在关键平台上
5. 包含2个紫色史莱姆作为守关者
6. 底部有安全网，掉落后可以重新开始攀登

---

## 备注
- **颗粒度**：3人日
- **美术需求**：PX（浮空平台、上升气流特效、安全网）、BG（云海背景）、VFX（风效果）
- **技术实现**：实现风力系统，上升气流区域检测
- **关卡设计**：
  - 起始区：安全练习区
  - 攀登区：逐步增加难度
  - 挑战区：强风+敌人组合
- **特殊机制**：
  - 风力系统：影响水平移动（风力50像素/秒）
  - 上升气流：持续向上推力（300像素/秒）
  - 安全网：底部区域，碰撞后传送到最近存档点
- **平台类型**：
  - 普通平台：静止，安全
  - 风力平台：受风影响，增加难度
- **敌人策略**：史莱姆放置在关键路径上
- **视觉提示**：上升气流有明显的视觉效果
- **性能要求**：风粒子效果不拖慢游戏，风力计算准确
- **存档系统**：每100米设置一个检查点
