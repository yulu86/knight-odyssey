# HUD界面设计

作为 一名玩家
我想要 拥有清晰美观的游戏界面显示重要信息
以便 随时了解游戏状态和进度

---

## 验收标准（AC）
1. 左上角显示生命值（3颗心形图标）
2. 右上角显示当前分数和金币数量
3. 底部中央显示关卡名称和进度条
4. 界面元素有半透明背景，不遮挡游戏画面
5. 分数增加时数字有放大缩小的动画效果
6. 生命值减少时心形图标有闪烁效果

---

## 备注
- **颗粒度**：1.5人日
- **美术需求**：UI（心形图标、数字字体、进度条、背景面板）、PX（界面装饰元素）
- **技术实现**：使用Control节点和CanvasLayer构建HUD
- **界面布局**：
  - 生命值：左上角，水平排列
  - 分数系统：右上角，垂直排列
  - 关卡信息：底部中央，水平布局
- **动画系统**：
  - 分数变化：Tween缩放动画（0.8-1.2-1.0）
  - 生命值闪烁：Shader实现呼吸效果
  - 进度条：SmoothStep插值更新
- **响应式设计**：适配不同分辨率
- **主题风格**：像素风格，与游戏整体风格一致
- **性能要求**：UI更新不影响游戏性能，60FPS
- **本地化**：支持中英文切换（预留功能）
- **可访问性**：重要信息使用高对比度颜色
