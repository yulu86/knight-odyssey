# 踩踏攻击机制

作为 一名玩家
我想要 从高处跳到敌人身上进行踩踏攻击
以便 能够消灭敌人并利用反弹跳得更高

---

## 验收标准（AC）
1. 当我从上方跳到敌人身上时，敌人被消灭
2. 踩踏成功后角色会向上反弹（反弹高度100像素）
3. 只有脚部接触敌人时才触发踩踏，侧面接触会受到伤害
4. 踩踏成功时有消灭敌人的视觉效果
5. 踩踏后有短暂的无敌时间（0.5秒）
6. 可以通过踩踏到达原本无法到达的高处

---

## 备注
- **颗粒度**：2人日
- **美术需求**：VFX（踩踏特效）、ANI（敌人消灭动画）、ANI（角色反弹动画）
- **技术实现**：使用Area2D检测脚部碰撞，通过velocity.y判断下落状态
- **碰撞检测**：设置专门的踩踏检测区域（foot_area）
- **反弹机制**：踩踏后设置velocity.y = -400
- **无敌时间**：使用Timer节点控制无敌持续时间
- **敌人反馈**：踩踏后播放消灭动画和音效
- **性能要求**：踩踏检测延迟<20ms
- **平衡调整**：可配置的踩踏高度和反弹力
