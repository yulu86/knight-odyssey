# 紫色史莱姆敌人

作为 一名玩家
我想要 遇到更强大的紫色史莱姆敌人
以便 面对更大的挑战并获得更好的游戏体验

---

## 验收标准（AC）
1. 紫色史莱姆有3点生命值，需要踩踏3次才能消灭
2. 移动速度为70像素/秒，比绿色史莱姆更快
3. 每次被踩踏后会变成更深的紫色，视觉上有明显变化
4. 被踩踏后短暂眩晕（1秒），然后继续巡逻
5. 紫色史莱姆消灭时掉落1枚金币
6. 紫色史莱姆的巡逻范围比绿色史莱姆更大（3米）

---

## 备注
- **颗粒度**：2人日
- **美术需求**：PX（紫色史莱姆精灵图）、ANI（移动动画、受伤动画、三种颜色状态）、VFX（眩晕效果）
- **技术实现**：继承自绿色史莱姆，添加生命值系统
- **AI行为**：增强的巡逻AI，添加受伤后的眩晕状态
- **视觉反馈**：根据生命值动态调整颜色（浅紫→中紫→深紫）
- **掉落机制**：消灭时使用信号通知生成金币
- **状态机**：巡逻→受伤→眩晕→巡逻
- **参数配置**：health = 3，speed = 70，stun_time = 1.0
- **性能要求**：每屏最多5个紫色史莱姆，优化碰撞检测
