# 绿色史莱姆敌人

作为 一名玩家
我想要 在关卡中遇到绿色史莱姆敌人
以便 增加游戏的挑战性和趣味性

---

## 验收标准（AC）
1. 绿色史莱姆在平台上左右巡逻移动
2. 移动速度为50像素/秒，移动范围为2米
3. 史莱姆有弹跳的移动动画
4. 被踩踏攻击时会消灭并消失
5. 触碰到玩家会扣除玩家1点生命值
6. 史莱姆碰到墙壁或悬崖边缘会自动转向

---

## 备注
- **颗粒度**：1.5人日
- **美术需求**：PX（史莱姆精灵图）、ANI（移动动画、消灭动画、弹跳效果）
- **技术实现**：使用CharacterBody2D，添加AI巡逻逻辑
- **AI行为**：简单的左右巡逻，添加raycast检测边缘
- **碰撞体**：设置适当的collision shape和hurt area
- **生命值**：生命值为1，一击必杀
- **音效**：SFX（移动弹跳声、消灭声）
- **参数配置**：speed = 50，patrol_distance = 200，damage = 1
- **性能要求**：每屏最多10个史莱姆，60FPS稳定运行
