# 基础跳跃机制

作为 一名玩家
我想要 按下空格键让角色跳跃
以便 能够越过障碍物和敌人

---

## 验收标准（AC）
1. 当我按下空格键时，角色执行跳跃动作
2. 跳跃高度为200像素，跳跃时间0.8秒
3. 角色在空中时只能进行左右移动，不能连续跳跃
4. 角色着地时有适当的着陆动画
5. 跳跃过程中播放跳跃动画
6. 角色跳跃时的抛物线轨迹自然流畅

---

## 备注
- **颗粒度**：1人日
- **美术需求**：PX（角色精灵图）、ANI（跳跃动画帧、着陆动画帧）
- **技术实现**：使用CharacterBody2D的gravity和jump_velocity参数
- **输入处理**：InputAction映射jump（默认空格键）
- **物理参数**：gravity = 1200，jump_velocity = -600
- **性能要求**：60FPS流畅运行，跳跃响应延迟<50ms
